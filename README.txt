-Devido a nuestra gran experiencia en lenguaje de marcas hemos decidido que el archivo que inicia el proyecto se llama index a si que porfavor para iniciar el proyecto ejecute el Frame Index

Requisitos previos:
-Isnatalción de la fuente de letra: Roboto 
-Disponible en \recursos\fonts\Roboto
-Puerto por defecto 3306
-Si al abrir el proyecto este contiene advertencias en lineas con "org." porfvor asegurese de añadir la libreria AbsoluteLayout.jar 
-Disponible en \lb\AbsoluteLayout.jar
-Si desea iniciar el programa sin haberse conectado a Xampp acepte las PopUps, el programa correra pero no guardara datos en la base de datos y saltaran la excepcion de intentar conectarse 
Recomendaciones:
-Porfavor agradeceriamos el uso con calma del programa a la hora de pulsar botones y secciones donde puedas escribir
